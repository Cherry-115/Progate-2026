このプロジェクトはハッカソンでのプロジェクトです。
チームメンバーは３人でフロント2人バック1人で開発しています。

以下のタスクはいかなる作業を行う場合でも必ず最初に行ってください。
1. Notion MCPを利用してこのプロジェクトに関するドキュメントやタスク管理を参照し現状と詳細な概要を確認し認識する。
あなたにはNotion MCPを利用する権限が与えられます。
ただし以下に示すNotionのページ以外へのアクセス、書き込みは一切禁止とします。
- root/ハッカソン チームホーム/*/
このパスの配下にこのプロジェクトの概要があります。
特に以下のパスが示すページには特に参照すべき概要が書かれているので必ず読んでください。
- root/ハッカソン チームホーム/ドキュメント/企画書*
- root/ハッカソン チームホーム/ドキュメント/要件定義書*
- root/ハッカソン チームホーム/ドキュメント/APIの仕様書
これらは必ず確認してください。

2. root/ハッカソン チームホーム/タスク管理 に行う作業の概要とログを残す。
この手順は実際に実装を行う場合にのみ適用されます。
もし、ドキュメントを生成するタスクを与えられた場合は以下のデータベースにドキュメントを追加してください。
- root/ハッカソン チームホーム/ドキュメント
これらはNotionデータベースとして作成されています。
特にタスク管理のページは以下の手順に従ってデータを追加してください。
  1. 登録されている全てのタスクを確認し行おうとしている作業と同様のものがないかを確認する
  2. なかった場合、このデータベースはsub-itemを利用し親子関係を表現しています
  行おうとしているタスクの親となるタスクを探してください
  3. 親となるタスクが見つかった場合そのタスクの子としてsub-itemを利用して追加してください
  もし親となるタスクが見つからなかった場合、親は設定せずにタスクを追加してください
  4. このデータベースにはさまざまなプロパティが設定されています
  特に以下のプロパティは必ず設定してください
    - カテゴリ
      - フロントエンド
      - バックエンド
      - インフラ
      - ドキュメント
      - その他
    
    - ステータス
      - 未着手
      - 進行中
      - レビュー待ち
      - 保留
      - 完了
    
    - メモ
      - String型　一行で簡潔にタスクの概要を書く
    
    - 優先度
      - 🔴高
      - 🟡中
      - 🟢低
